## How to run

### Install required modules
```sh
pip install flask flask_sqlalchemy flask_login
```

### Run applicaion
```sh
python -m flask --app src/flaskr run
```
### Datenbank initialisieren
```sh
python -m flask --app src/flaskr init-db
```
